## v1.0-pre-refactor
- Stable NewsApp using NewsAPI
- Category-based news (sports, science, technology, business, etc.)
- Pagination implemented using API-driven (server-side) pagination via NewsAPI page and pageSize parameters. (18 articles per page)
- Functional Next / Previous navigation
- News cards include source, author, and published date
